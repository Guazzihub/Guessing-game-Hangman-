 O jogo da forca me mostrou o quanto eu avancei em questão a lógica de programação durante o 1º semestre de 2019. Inicialmente seria apenas um jogo simples para entrega de projeto, permitindo vários erros como: entrar com vários caracteres, entrar com espaço vazio etc. Mas tomei como desafio fazer com que o programa tivesse o mínimo de erros possível. Em geral foi um projeto de faculdade muito divertido de se desenvolver e também aprendi muito, além de praticar tudo o que já havia aprendido durante o semestre. 

 Estou ciente de que este programa poderia ser aprimorado e otimizado, mas por enquanto gostaria de praticar outros projetos. Caso encontre algum erro no código por favor fique a vontade para comentar sobre a falha, ficarei feliz de checar.
