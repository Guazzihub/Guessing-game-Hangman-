 O jogo da forca me mostrou o quanto eu avancei em questão a lógica e programação durante o 1º semestre de 2019. Inicialmente seria apenas um jogo simples para entrega 
de projeto, permitindo vários erros como entrar com vários caracteres, entrar com espaço vazio etc. Mas tomei como um desafio fazer com que o programa tivesse o mínimo
de erros possível. Em geral foi um projeto de faculdade muito divertido de se fazer e também aprendi muito, além de praticar tudo o que já havia aprendido durante o 
semestre obtive novos conhecimentos. 

 Estou ciente de que este programa poderia ser aprimorado e otimizado, mas por enquanto gostaria de praticar outros projetos. Caso encontre algum erro no código por 
favor fique a vontade para comentar sobre, ficarei feliz de checar o erro.
 
Rev2: Quando uma nova palavra é gravada o programa não fecha mais.
